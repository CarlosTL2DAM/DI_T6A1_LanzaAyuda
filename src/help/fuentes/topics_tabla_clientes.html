Tabla de clientes:



En esta tabla se muestran los distintos clientes almacenados en la base de datos, donde se exponen los siguientes datos del cliente:


→ DNI: Muestra el DNI del cliente.



→ Nombre: Representa el nombre del cliente.



→ Dirección: Representa la dirección donde vive el cliente.



→ Localidad: Indica la localidad de la que procede el cliente. Este apartado puede encontrarse vacío en caso de que el cliente solo haya realizado reservas del salón Habana.




→ Provincia: Indica la provincia en la que vive el cliente. Este apartado, al igual que el anterior puede encontrarse vacío en caso de que el cliente solo haya realizado reservas del salón Habana.




→ Teléfono: Muestra información sobre el teléfono de contacto del cliente. Este apartado puede encontrarse vacío en caso de que el cliente solo haya realizado reservas de habitaciones.
